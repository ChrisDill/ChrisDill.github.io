Sirius game

My main project I am working on with a friend in my spare time. Its a topdown sci-fi shooter inspired by games such as FTL - Faster than light and Heat signature. I have a devlog serious about it on my blog. It is being custom made in c# using raylib-cs.


Open source

Made a binding of raylib, a simple and easy to use library for video games programming. Wanted to try using it with c# for a game so I made a binding. Learned alot of useful skills such as project management, pull requests and forks.


Mech shooter

College project as part of a team. Gave myself a goal to make a prototype in Unreal for a first person shooter but with mech. Used the blueprint system for the logic and learned the basics of lower level c++. End result had 2 player controlled mechs on a platform that could fire at each other until only one is left.
